# Collection de RPG Navigateur

Bienvenue dans **RPG sur navigateur**, un petit jeux de rôle jouables directement depuis votre navigateur web. des quêtes à accomplir et des combats stratégiques.

## Description
Jeu RPG jouables sur navigateur, créés pour explorer différents styles de gameplay :  
- **RPG dynamique avec coldown sur les spell**  
- **RPG d’aventure avec exploration de cartes**  
- **Mini-jeux RPG avec quêtes et combats rapides**  

Chaque projet est autonome et peut être ouvert dans un navigateur moderne sans installation complexe.

## Projets inclus
| Nom du projet | Description courte | URL / Fichier |
|---------------|-----------------|---------------|
| **AventureFantastique** | RPG d’exploration avec quêtes et dialogues | `AventureFantastique/index.html` |
| **CombatMystique** | RPG au tour par tour avec ennemis aléatoires | `CombatMystique/index.html` |
| **MiniQuêtes** | Collection de petites missions RPG rapides | `MiniQuêtes/index.html` |

*(Ajoutez ou modifiez les projets selon votre collection)*

## Caractéristiques communes
- Jouable directement dans un navigateur (Chrome, Firefox, Edge)
- Sauvegarde de la progression (localStorage)
- Combat et progression de personnage
- Quêtes principales et secondaires
- Interface simple et intuitive
